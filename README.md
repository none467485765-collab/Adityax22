# SSMV School Website

Modern React + Vite + Tailwind CSS school website.

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build For Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run start
```

## Render Deployment

Build Command:

```bash
npm install && npm run build
```

Publish Directory:

```txt
dist
```
